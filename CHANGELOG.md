## [1.0.0] - 2024-03-19

### Added
- Initial release
- Automatic attachment organization by file type
- Support for paste and drop events
- WikiLinks and Markdown link format options
- Auto-rename duplicate files
- Desktop and mobile support
- File type categorization (images, audio, video, documents, archives, code)
- Settings tab for configuration
